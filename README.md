# 个人时间管理系统

一个基于 React + TypeScript 的个人时间管理网页应用，帮助您高效管理学习、工作和生活时间。

## 功能特性

### 🗂️ 分类管理
- 创建和管理大分类（如：技术书籍、闲书、项目、学习等）
- 每个分类下可以创建多个具体项目
- 支持自定义分类颜色和描述

### 📝 时间记录
- 为每个项目记录时间花费
- 支持按日期记录时间
- 添加备注信息
- 查看项目总时间统计

### 📊 统计分析
- **日统计**：当天时间分布
- **周统计**：本周时间趋势  
- **月统计**：月度时间汇总
- **年统计**：年度时间概览
- **自定义日期范围**：灵活选择统计时间段

### 📈 数据可视化
- 时间趋势折线图
- 分类分布饼图
- 项目分布柱状图
- 项目进度追踪

### 💾 数据管理
- 本地存储，数据持久化
- 数据导入导出功能
- 自动备份和恢复

## 技术栈

- **前端框架**: React 18 + TypeScript
- **状态管理**: React Hooks + Context API
- **样式方案**: Tailwind CSS
- **数据可视化**: Recharts
- **日期处理**: date-fns
- **构建工具**: Vite

## 使用说明

### 1. 创建分类
- 点击"添加分类"按钮
- 输入分类名称、描述和选择颜色
- 系统已预设常用分类：技术书籍、闲书、项目、学习

### 2. 添加项目
- 点击"添加项目"按钮
- 选择所属分类
- 输入项目名称、描述、目标小时数和颜色
- 系统已预设示例项目：CPP开发实战、CPP并发编程实战等

### 3. 记录时间
- 在"时间记录"页面选择项目
- 点击"记录时间"按钮
- 输入日期、小时数和备注
- 查看项目的时间记录列表

### 4. 查看统计
- 在"统计分析"页面查看各种统计图表
- 切换不同的时间周期（日、周、月、年、自定义）
- 查看项目进度和完成情况

### 5. 数据管理
- 使用"导出数据"按钮备份数据
- 使用"导入数据"按钮恢复数据

## 最佳实践

1. **分类设计**：建议按学习领域、工作项目、兴趣爱好等维度创建分类
2. **项目命名**：使用具体、明确的名称，便于识别和统计
3. **时间记录**：养成每日记录的习惯，及时记录时间花费
4. **目标设定**：为重要项目设定目标小时数，跟踪完成进度
5. **定期回顾**：利用统计功能定期回顾时间分配，优化时间管理
